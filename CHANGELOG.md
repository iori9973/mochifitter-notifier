# Changelog

## [1.0.0] - 2026-02-22

### Added
- もちふぃった～（OutfitRetargetingSystem）の処理完了を検知して通知音を再生
- デフォルト通知音（7音メロディ）を内蔵（外部ファイル不要）
- `Edit → Preferences → MochiFitter Notifier` から設定可能
  - 通知の ON/OFF 切り替え
  - カスタム音声ファイルの指定（WAV / MP3 / M4A）
  - テスト再生ボタン
- VCC（VRChat Creator Companion）によるインストールに対応
- `.unitypackage` によるインストールに対応
- Windows 対応
